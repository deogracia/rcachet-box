# rcachet-box
my dev env :D

## inhalt
 * Debian 8 64b
 * cachet v1.2.1 (in /home/vagrant/Cachet)

## setup

```
git clone git@github.com:deogracia/rcachet-box.git
cd rcachet-box
git clone https://bitbucket.org/deogracia/provisionning.git
git clone https://github.com/deogracia/rcachet

vagrant up
```
