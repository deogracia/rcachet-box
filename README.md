# rcachet-box
my dev env :D

## inhalt

 * cachet v1.2.1 (in /home/vagrant/Cachet)

## setup

```
git clone git@github.com:deogracia/rcachet-box.git
cd rcachet-box
git clone git@bitbucket.org:deogracia/provisionning.git
git clone git@github.com:deogracia/rcachet.git

vagrant up
```

That's all :D
